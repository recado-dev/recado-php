# Contributing

Thanks for your interest in `recado/recado-php`! 🙌

## How this repo works

This package is developed inside a **private monorepo**, which is the single
source of truth. **This public repository is a read-only mirror**, regenerated
automatically (force-pushed) on every change to the source — so anything
committed directly here is overwritten on the next sync.

Because of that:

- **Pull requests opened here are automatically closed.** They can't be merged
  into a force-pushed mirror, and they can't reach the private source. This is
  not a judgement on your change — the mirror simply isn't a place code can land.

## How to contribute

**Open an issue** on this repo for bugs, questions or feature requests. Issues
are enabled and watched — the maintainers triage them and port accepted fixes
into the private source, which then flows back out to this mirror automatically.

When filing an issue, the usual helpers go a long way:

- the SDK version (or `composer show recado/recado-php`) and your PHP version,
- a minimal reproduction or the exact call you made,
- what you expected vs. what happened (including any exception `code`/message).

Thanks for helping make the SDK better! 💛
